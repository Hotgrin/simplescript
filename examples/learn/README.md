# Learn hotgrin, one small program at a time

A numbered path from your very first line to complete mini-projects. Every
file is a runnable, heavily-commented program — read it, run it, change it,
break it, fix it. That cycle IS learning to program.

```bash
hotgrin run 01-say-hello.hot
```

**Never coded before?** Start at **[Day Zero](../../docs/day-zero.md)** — a
five-minute, no-computer page, before lesson 01.
